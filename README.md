# Сайт «ЮГ ЕвроИнвест»

Лендинг поставщика металлопроката (Алматы / Астана) с каталогом, корзиной без цен,
формами заявок и интеграцией с Google Таблицей.

## Состав проекта

Сайт **многостраничный**. Общие шапка, подвал, корзина и модалки
вставляются скриптом `app.js` на каждой странице (правите в одном месте).

| Файл | Назначение |
|------|------------|
| `index.html` | Главная (hero, преимущества, этапы, CTA) |
| `catalog.html` | Каталог (категории → карточки товаров) |
| `company.html` | Компания (о компании, преимущества) |
| `services.html` | Услуги |
| `contacts.html` | Контакты (телефон, email, форма, карта) |
| `styles.css` | Стили (тёмно-синий + золото, анимации, адаптив) |
| `app.js` | Общий скрипт: шапка/подвал/корзина/формы/данные каталога/анимации |
| `catalog.js` | Логика страницы каталога (категории и карточки товаров) |
| `google-apps-script/Code.gs` | Скрипт приёма заявок в Google Таблицу |
| `assets/IMAGES.md` | **Список всех картинок и их имён** (как менять фото) |
| `assets/logo.png` | Логотип компании (нужно добавить, см. ниже) |

> Каталог и его данные (категории, товары, размеры, имена картинок)
> задаются массивом `CATEGORIES` в начале `app.js`.

## Быстрый запуск (локально)

Откройте папку и запустите простой сервер (картинки и формы любят `http://`, а не `file://`):

```powershell
# Node.js (рекомендуется)
node server.js
```

Затем откройте `http://localhost:8123` в браузере.
Можно также через Python: `python -m http.server 5500`.

## 1. Логотип

Сохраните присланный логотип как `assets/logo.png`.
Если файла нет — текстовый логотип «ЮГ ЕвроИнвест» отобразится автоматически.

## 2. Настройка Google Таблицы (приём заявок)

1. Откройте [Google Диск](https://drive.google.com) → создайте новую **Google Таблицу**
   (например, «ЮГ ЕвроИнвест — Заявки»).
2. В меню таблицы: **Расширения → Apps Script**.
3. Удалите содержимое файла `Код.gs` и вставьте код из `google-apps-script/Code.gs`.
   - (Необязательно) в строке `var MANAGER_EMAIL = "";` укажите email менеджера —
     ему будут приходить уведомления о новых заявках.
4. Нажмите **Сохранить**.
5. Нажмите **Развернуть → Новое развёртывание**.
   - Тип: **Веб-приложение**.
   - «Кто имеет доступ»: **Все** (Anyone).
   - Нажмите **Развернуть**, разрешите доступ к аккаунту.
6. Скопируйте полученный **URL веб-приложения** (вида `https://script.google.com/macros/s/.../exec`).
7. Откройте `app.js` и вставьте URL в строку:

   ```js
   const GOOGLE_SCRIPT_URL = "ВСТАВЬТЕ_СЮДА_URL";
   ```

Готово. Теперь:
- Заказы из корзины → лист **«Заказы»**.
- Формы «Заказать звонок» и «Рассчитать стоимость» → лист **«Заявки на звонок»**.
- Если клиент указал email — ему уйдёт письмо-подтверждение.

> Листы и заголовки столбцов создаются автоматически при первой заявке.

### Столбцы листа «Заказы»
Дата/время · ФИО · Телефон · Email · Город · Список товаров · Комментарий · Статус

### Столбцы листа «Заявки на звонок»
Дата/время · Имя · Телефон · Город · Статус

Колонку **Статус** менеджеры заполняют вручную (Новый / В работе / Обработан).

## 3. Проверка интеграции

1. Откройте сайт, добавьте товар в корзину, оформите заказ.
2. Обновите Google Таблицу — должна появиться новая строка.

Если URL не задан, заявки выводятся в консоль браузера (F12) — удобно для теста вёрстки.

## 4. Деплой на хостинг

Это статический сайт — подойдёт любой хостинг: Netlify, Vercel, GitHub Pages,
обычный хостинг с папкой `public_html`. Просто загрузите все файлы.

## Изменение каталога

Каталог задаётся массивом `CATEGORIES` в начале `app.js`.
Каждая категория: `name`, `img` (локальный путь к фото), `fallback` (резерв), `desc`,
`products` (подвиды с `slug` для имени картинки, `unit` и `sizes`).

Имена и расположение всех картинок — в `assets/IMAGES.md`.
